# Product Demand Analysis System using SAP ABAP

## Description
This project is an analytical reporting system built using SAP ABAP. It generates product data and performs demand analysis.

## Features
- Data generation (200 products)
- Top 5 product ranking
- Inventory classification
- Category-wise analysis

## Tech Stack
- SAP ABAP (Eclipse ADT)
- Internal Tables

## Output
- Summary
- Inventory Status
- Top 5 Products
- Category Analysis
