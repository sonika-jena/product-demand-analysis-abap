# Product Demand Analysis System using SAP ABAP

## Description
This project is a database-driven analytical reporting system developed using SAP ABAP. It stores product data in a custom database table and performs demand analysis to generate meaningful business insights.

## Features
- Database-based data storage (ZPROD_ANALYSIS)
- One-time data generation using ABAP
- Category and subcategory mapping
- Total sales calculation
- Top 5 product ranking
- Inventory classification (Restock / Overstock / Normal)
- Category-wise analysis (total sales, top product, max sales)

## Tech Stack
- SAP ABAP (Eclipse ADT)
- Open SQL (SELECT, INSERT)
- Internal Tables

## Output
- Summary (Total Sales, Top Product, Max Sales)
- Inventory Status
- Top 5 Products
- Category-wise Analysis
